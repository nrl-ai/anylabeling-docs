# Auto Use Last Label

If you need to annotate objects with a same label repeatedly, you can enable **Auto Use Last Label** from the **Edit** menu. In this mode, new objects will be annotated with the last label automatically, including the objects created by the **Auto Labeling** tool.

![Auto Use Last Label](/images/auto-use-last-label.png)

When you need to update the label, select the **Edit** mode, edit the label of the last annotated object. The label of the last annotated object will be used for the next annotations.

![Auto Use Last Label](/images/switch-label.png)
