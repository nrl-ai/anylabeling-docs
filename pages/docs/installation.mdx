# Installation

There are three ways to install AnyLabeling.

## 1. Install from PyPI

```bash
pip install anylabeling
```

Or with GPU support:

```bash
pip install anylabeling-gpu
```

Run AnyLabeling:

```bash
anylabeling
```

## 2. Install from binary distribution

Download the latest binary distribution from [Releases](https://github.com/vietanhdev/anylabeling/releases).

**Note:** Choose versions with `-GPU` for GPU support.

## 3. Install from source

```bash
git clone https://github.com/vietanhdev/anylabeling
cd anylabeling
pip install .
```

Run AnyLabeling:

```bash
anylabeling
```
